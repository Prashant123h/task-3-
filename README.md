# Personal Portfolio Website

This project is an interactive personal portfolio website built using HTML, CSS, and JavaScript. It was developed as part of my Web Development Internship at Maincrafts Technology.

The goal of this project was to enhance a static portfolio website by adding dynamic features and improving user interaction using JavaScript and DOM manipulation.

## Features

• Responsive modern portfolio layout  
• Dynamic navigation bar that changes on scroll  
• Active navigation highlighting for sections  
• Contact form with JavaScript validation  
• Email format validation  
• Dark Mode / Light Mode toggle  
• Theme preference saved using LocalStorage  

## Technologies Used

HTML5  
CSS3  
JavaScript (DOM Manipulation)
